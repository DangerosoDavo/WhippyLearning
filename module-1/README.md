# Learning to Code: From Zero to Chat App

Welcome Bret! This is your coding journey. By the end of these lessons, you'll have built a working chat application from scratch.

## How to Use These Lessons

1. Go through each lesson **in order** — they build on each other
2. Each lesson has a **lesson notes** file explaining concepts and a **hands-on exercise** file
3. Open the `.html` exercise files in your web browser (just double-click them)
4. Edit the `.html` files using a text editor (VS Code, Notepad++, or even Notepad)
5. After making changes, refresh your browser to see the results

## Lesson Plan

| # | Lesson | What You'll Learn |
|---|--------|-------------------|
| 1 | Your First Webpage | What HTML is, basic tags, seeing results in a browser |
| 2 | Styling with CSS | Making things look good — colors, fonts, layout |
| 3 | Hello JavaScript | Variables, math, printing to the console |
| 4 | Talking to the Page | Using JS to read and change what's on the page |
| 5 | Clicks and Events | Making buttons and inputs actually do things |
| 6 | Making Decisions | If/else logic, functions, organizing your code |
| 7 | Lists of Things | Arrays, loops, and working with collections of data |
| 8 | Building the Chat UI | Creating the visual layout for our chat app |
| 9 | Making Chat Work | JavaScript to send, display, and manage messages |
| 10 | Finishing Touches | Timestamps, usernames, saving messages, polish |

## What You'll Need

- A **web browser** (Chrome, Firefox, or Edge)
- A **text editor** (VS Code is recommended — it's free)
- Curiosity and patience!

Happy coding!
