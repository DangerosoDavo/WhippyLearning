# WhippyLearning

Hey Bret! This repo has everything you need to learn how to code. Each module is a set of lessons that build on each other, taking you from zero to building real projects.

## Getting Started

### Step 1: Download this repo to your computer

1. Open a terminal (search for **Git Bash** in your Start menu)
2. Navigate to where you want to save the project. For example, to save it on your Desktop:
   ```
   cd ~/Desktop
   ```
3. Clone (download) the repo:
   ```
   git clone https://github.com/DangerosoDavo/WhippyLearning.git
   ```
4. You should now see a folder called `WhippyLearning` on your Desktop

### Step 2: Open the folder in VS Code

1. Open **VS Code**
2. Go to **File > Open Folder...**
3. Navigate to the `WhippyLearning` folder you just downloaded and click **Select Folder**

That's it! You should see all the lesson folders in the sidebar on the left.

### Step 3: Start learning

1. Open the `module-1` folder in the sidebar
2. Read the `README.md` inside it for the full lesson plan
3. Start with `lesson-01-your-first-webpage` — open `notes.md` to learn the concepts, then open `exercise.html` to practice
4. To see your exercises in action, right-click the `exercise.html` file in VS Code and choose **Open with Live Server** (if you have the extension) or just find the file in File Explorer and double-click it to open in your browser

### How each lesson works

- **notes.md** — Read this first. It explains new concepts with examples.
- **exercise.html** — Open this in your text editor AND your browser. Edit the code, save, and refresh the browser to see your changes.

### Getting updates

If new lessons or modules are added, you can pull the latest version. Open Git Bash, navigate to the folder, and run:

```
cd ~/Desktop/WhippyLearning
git pull
```
